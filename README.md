# lu-shui

Hello my name is Luise (pronouns "she/her").

This is a Nuxt 3-based portfolio and documentation site showcasing archaeological findings and research.

## Setup

To set up and run this project locally, follow these steps:

1. **Clone the repository:**
   ```
   git clone https://github.com/your-username/lu-shui.git
   cd lu-shui
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the development server:**
   ```
   npm run dev
   ```
   The application will be available at `http://localhost:3000`.

4. **Build for production:**
   ```
   npm run build
   ```

5. **Preview the production build:**
   ```
   npm run preview
   ```

### Additional Commands

- **Lint the code:** `npm run lint`
- **Fix linting issues:** `npm run lint:fix`
- **Generate static site:** `npm run generate`

### Requirements

- Node.js (version specified in `.nvmrc`)
- npm

### Notes

This project uses Nuxt 3 with the Content module for managing content. Node.js version is managed via `.nvmrc` for consistency across environments.
